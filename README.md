# NeuronSpectrum

Python-проект для работы с EEG через LSL (Lab Streaming Layer).

Проект ориентирован на поток EEG из Нейроспектр.NET и объединяет инструменты для записи, анализа и сравнения корреляционных матриц между каналами. Основной сценарий работы включает поиск доступных LSL-потоков, запись сессии, offline-анализ сохранённых данных, live-корреляции по скользящему окну и сравнение ранее рассчитанных результатов.

Типичный поток в проекте:

- 20 каналов
- 500 Гц
- окно анализа около 2 секунд
